Overview

The AutoSol Enterprise Server (AES) is software that communicates with different manufacturer SCADA and utility metering devices in their native protocols, over a single or multiple telemetry circuits. Simultaneously, AES delivers real-time data to SCADA host systems and historical meter data to EFM/AMR data validation and accounting systems, using a single configurable solution.

Scalability & Redundancy

The AES runtime and configuration databases permit online changes while maintaining continuous operations, providing non-stop data acquisition for safety and business critical applications. Additionally, AES supports redundancy at several levels, including redundant communication paths, redundant configurations, and load sharing, all of which allow automatic switch-over upon detection of a failure.This scalability provides you with the flexibility to expand your system on a just-in-time basis further maximizing your return on investment. These are just a few of the industry leading capabilities of the AutoSol Enterprise Server (AES).
